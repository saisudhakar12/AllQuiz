🔴 PROBLEM: "Cannot load topic file : TypeError"

═════════════════════════════════════════════════════════════════════════════════

🔍 ROOT CAUSE:
   You are trying to open quiz.html directly from Windows Explorer (file:// protocol)
   instead of using a local HTTP server.

   Browser Security:
   • file:// protocol blocks fetch() requests
   • We use fetch() to load quiz data from data/ folder
   • This causes TypeError: failed to fetch

═════════════════════════════════════════════════════════════════════════════════

✅ SOLUTION: Use a Local Server

═════════════════════════════════════════════════════════════════════════════════

🟢 BEST WAY: Double-Click to Start Server

   1. Find this file in your quiz project folder:
      START_SERVER.bat

   2. Double-click it
   3. PowerShell opens with server running
   4. You'll see: "Serving HTTP on 0.0.0.0 port 8000"
   5. Browse to: http://localhost:8000
   Done! ✅

═════════════════════════════════════════════════════════════════════════════════

🟡 ALTERNATIVE: Manual Command (PowerShell)

   1. Open PowerShell
   2. Run this command:
      cd C:\Users\sudha\IdeaProjects\TestProject

   3. Then run:
      python -m http.server 8000

   4. Browse to: http://localhost:8000
   Done! ✅
